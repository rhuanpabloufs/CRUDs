########################  USUARIO  ########################

--- CREATE (201/200) ---
curl -X POST http://localhost:8080/usuario -H "Content-Type: application/json" -d "{\"cpf\": 12345678901, \"nome\": \"Joao Silva\", \"login\": \"joaotest\", \"senha\": \"123\", \"email\": [\"joao@email.com\"], \"telefone\": [\"79999998888\"]}"

--- READ todos ---
curl http://localhost:8080/usuario

--- READ por id ---
curl http://localhost:8080/usuario/12345678901

--- UPDATE (nome + array email sem duplicar) ---
curl -X PUT http://localhost:8080/usuario/12345678901 -H "Content-Type: application/json" -d "{\"nome\": \"Joao Atualizado\", \"email\": [\"joao@email.com\", \"novo@email.com\"]}"
curl http://localhost:8080/usuario/12345678901

--- DELETE (retorna objeto, 200) ---
curl -X DELETE http://localhost:8080/usuario/12345678901

--- HANDLER 404 (nao encontrado) ---
curl http://localhost:8080/usuario/00000000000

--- HANDLER 409 (login duplicado) -> rode o MESMO create 2x ---
curl -X POST http://localhost:8080/usuario -H "Content-Type: application/json" -d "{\"cpf\": 11111111111, \"nome\": \"Dup\", \"login\": \"repetido\", \"senha\": \"1\"}"
curl -X POST http://localhost:8080/usuario -H "Content-Type: application/json" -d "{\"cpf\": 22222222222, \"nome\": \"Dup2\", \"login\": \"repetido\", \"senha\": \"1\"}"

--- HANDLER 400 (campo obrigatorio faltando / Bean Validation) ---
curl -X POST http://localhost:8080/usuario -H "Content-Type: application/json" -d "{\"cpf\": 33333333333, \"login\": \"semnome\", \"senha\": \"1\"}"

--- HANDLER 400 (JSON malformado) ---
curl -X POST http://localhost:8080/usuario -H "Content-Type: application/json" -d "{\"cpf\": 44444444444, \"nome\": \"X\" "


########################  CURSO  ########################

--- CREATE (valor com espaco - o caso dificil) ---
curl -X POST http://localhost:8080/cursos -H "Content-Type: application/json" -d "{\"nome\": \"Teste Lic\", \"grau\": \"Licenciatura Plena\", \"turno\": \"Turno Indefinido\", \"campus\": \"Itabaiana\", \"nivel\": \"Lato\"}"

--- READ todos ---
curl http://localhost:8080/cursos

--- READ por id (ajuste o id) ---
curl http://localhost:8080/cursos/6

--- UPDATE parcial (so campus, mantem resto) ---
curl -X PUT http://localhost:8080/cursos/6 -H "Content-Type: application/json" -d "{\"campus\": \"Aracaju\"}"
curl http://localhost:8080/cursos/6

curl -X PUT http://localhost:8080/cursos/6 -H "Content-Type: application/json" -d "{\"turno\": \"Noturno\"}"
curl http://localhost:8080/cursos/6

--- DELETE ---
curl -X DELETE http://localhost:8080/cursos/6

--- HANDLER 404 ---
curl http://localhost:8080/cursos/9999

--- HANDLER 409 (combinacao duplicada) -> rode o MESMO create 2x ---
curl -X POST http://localhost:8080/cursos -H "Content-Type: application/json" -d "{\"nome\": \"Repetido\", \"grau\": \"Bacharelado\", \"turno\": \"Matutino\", \"campus\": \"Lagarto\", \"nivel\": \"Graduação\"}"
curl -X POST http://localhost:8080/cursos -H "Content-Type: application/json" -d "{\"nome\": \"Repetido\", \"grau\": \"Bacharelado\", \"turno\": \"Matutino\", \"campus\": \"Lagarto\", \"nivel\": \"Graduação\"}"

--- HANDLER 400 (nome vazio / Bean Validation) ---
curl -X POST http://localhost:8080/cursos -H "Content-Type: application/json" -d "{\"nome\": \"\", \"grau\": \"Bacharelado\", \"turno\": \"Vespertino\", \"campus\": \"X\", \"nivel\": \"Lato\"}"

--- HANDLER 400 (enum invalido - banco rejeita no cast) ---
curl -X POST http://localhost:8080/cursos -H "Content-Type: application/json" -d "{\"nome\": \"Y\", \"grau\": \"Bacharelado\", \"turno\": \"NaoExiste\", \"campus\": \"X\", \"nivel\": \"Lato\"}"

RESUMO DOS HANDLERS QUE DEVEM SER ACIONADOS:

- 409 DadosDuplicado ........ login repetido, combinacao de curso repetida
- 400 Bean Validation ....... campo obrigatorio faltando, nome vazio
- 400 JSON malformado ....... JSON quebrado (HttpMessageNotReadable)
- 400/integridade .......... enum invalido (cast falha), FK inexistente
